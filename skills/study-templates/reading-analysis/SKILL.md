# Reading Analysis Template

Use this Skill for Hermes Mobile reading or retelling study cards.

- Use the current transcript as primary evidence.
- Return learner-facing Chinese Markdown.
- Do not claim acoustic pronunciation evidence unless transcription metadata supports it.
- Include score, deductions, evidence, comprehension, retelling quality, English expression or grammar, vocabulary and sentence patterns, comparison with previous sessions, next-session advice, and parent observation points.
- Quiz targets must come from the transcript and analysis, not unrelated grammar or vocabulary drills.
