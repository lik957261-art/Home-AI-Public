# English Listening Input Template

Use this template only through Hermes Mobile Growth services.

## Learner Flow

1. Listen to the assigned short input.
2. Write 3-5 English key points.
3. Review AI gap feedback.
4. Retry the missed part.
5. Receive evaluation and next-practice guidance.

## Feedback Contract

Feedback must identify missing key points, detail accuracy, and the exact retry target. It must include a small repair task rather than only a score.

## Privacy

Do not store full audio transcripts or raw recordings in model-facing summaries. Store only key-point summaries, gap summaries, retry summaries, score, and report references.
