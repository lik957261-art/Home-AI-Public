# General Assessment Template

Use this Skill for Hermes Mobile formal assessment cards that are not using a more specific template.

- Generate original single-answer multiple-choice questions.
- Do not copy copyrighted exam questions.
- Keep each question internally consistent, with four choices, one correct answer, a concise skill tag, and a useful explanation.
- Final reports should put score, wrong items, weak points, and review advice before any per-question detail.
- Use Chinese by default for learner-facing reports unless the card explicitly asks for another language.
