# English Weekly Challenge Template

Use this template only through Hermes Mobile Growth services.

## Learner Flow

1. Review this week's cleaned progress signals.
2. Complete one integrated English task using reading, vocabulary, and expression repairs.
3. Improve one sentence.
4. Add one reflection sentence.
5. Receive final evaluation, report, and reward settlement.

## Feedback Contract

Feedback must reference the week's summary-level learning signals, not raw child content. It must explain integration, accuracy, transfer of repairs, and next-week focus.

## Privacy

Store only weekly signal summaries, integrated answer summaries, repair summaries, score, reward status, and report references. Do not store full answers, transcripts, question text, answer keys, raw prompts, local paths, or secrets.
