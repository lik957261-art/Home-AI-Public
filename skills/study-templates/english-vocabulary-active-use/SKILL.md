# English Vocabulary Active Use Template

Use this template only through Hermes Mobile Growth services.

## Learner Flow

1. Write five original English sentences using target vocabulary.
2. Receive AI feedback on meaning, context fit, and sentence accuracy.
3. Repair at least two sentences.
4. Receive evaluation and next-practice guidance.

## Feedback Contract

Feedback must state whether each target word is used with the right meaning and context, then require concrete repairs. It must include at least one improved example.

## Privacy

Store only sentence summaries, repair summaries, score, and report references. Do not store full child answers, full prompts, local paths, or secrets.
